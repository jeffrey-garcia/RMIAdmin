Export Debug Console
RMIAdmin allows you to export the debug console.


The debug console of RMIAdmin could be exported (in .LOG format). This is very useful for error tracing as well as activity logging, moreover it also help us to diagnose the cause of errors reported/received.

Usage instructions
  1. On the toolbar of RMIAdmin, select "File" > "Export" > "Debug Console".
  2. Simply choose the target location and key-in the filename.
  3. Click "Save" to proceed, click "Cancel" to abort.
