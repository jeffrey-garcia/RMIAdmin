Export Server List
RMIAdmin allows you to export the managed server list.


The managed server list of RMIAdmin could be exported (in XML format), such that you could export the server list and use it on other RMIAdmin client.

Usage instructions
  1. On the toolbar of RMIAdmin, select "File" > "Export" > "Server List".
  2. Simply choose the target location and key-in the filename.
  3. Click "Save" to proceed, click "Cancel" to abort.
