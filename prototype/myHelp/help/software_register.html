Software Registration
RMIAdmin is a licensed protected software but allows 30 Days free evaluation. After that the software will expire.


Purchase
To purchase RMIAdmin, please read http://www.rmiAdmin.net -> Purchase (Require Direct Internet Access)


Support
If any assistance is required, please goto http://www.rmiAdmin.net -> Support (Require Direct Internet Access), or
send an email to cs@rmiadmin.net


If you have purchased the registration key, simply complete the registration key as below:
  1. From the toolbar, select "File" -> "Register"
  2. You will then see the registration screen, simply enter the license, the click "OK" to continue, "Cancel" to abort.
  3. If the registration success, you will then see the below screen.
  4. Only when you've completed the registration, the "Generate Authentication" feature will be enabled.
    See Generate Authentication for details.
