What is covered in this help menu?

RMIAdmin Help explains how to use the RMIAdmin Server and RMIAdmin Client, introducing Administrators / Developers / Users to RMIAdmin features and common tasks.

Major functions including Remote Command Execution, File Transfer Tool, Remote File System Browsers, Link Connection and Virtual Machine Memory Monitor, and Graphical Remote Control (Beta).

It favours administration by supporting the creation of customized batch job, shell scripts, or integrating RMIAdmin with other software.
