Import Server List
RMIAdmin allows you to import managed server list previously created.


The managed server list of RMIAdmin could be imported (in XML format), such that you could import the server list which is previously being use by other RMIAdmin client.

Usage instructions
  1. On the toolbar of RMIAdmin, select "File" > "Import" > "Server List".
  2. Simply navigate to the target location and choose the preferred file.
  3. Click "Open" to proceed, click "Cancel" to abort.
